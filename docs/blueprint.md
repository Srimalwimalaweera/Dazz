# **App Name**: Dazz

## Core Features:

- Navbar: Sticky navbar with logo, links (Home, Services, Gallery, Contact), and a WhatsApp Us button.
- Hero Section: High-impact hero section with featured image, headline, sub-headline, and CTA button to WhatsApp.
- Our Services: Grid layout of 3 cards: Premium Mugs, Custom T-Shirts, Magic Prints, with hover effects and descriptions.
- The Experience: About Us section explaining the DAZZ Vibe, quality, packaging, and unboxing experience.
- Footer: Footer with square logo, links to social media, contact info, and copyright text.
- WhatsApp Integration: Direct links to WhatsApp for 'Start Creating' CTA and support.
- Responsive Design: Fully responsive design for optimal viewing on all devices.

## Style Guidelines:

- Primary color: Silver (#C0C0C0) for a luxurious feel.
- Background color: Deep Matte Black (#050505) providing a premium, dark backdrop. Enforce Dark Theme only. Do not include any Light/Dark mode toggles. Ignore user system preferences for light mode.
- Accent color: Gold (#FFD700) used sparingly for highlights and call-to-action elements to create contrast and attract attention.
- Body and headline font: 'Inter' (sans-serif) for clean, modern readability. Note: currently only Google Fonts are supported.
- Use minimalist icons for services and social media links.
- Use a grid layout for services to present them in a structured, visually appealing manner.
- Smooth scroll effect and subtle hover animations on service cards.
